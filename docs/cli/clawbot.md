# `lada clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `lada clawbot qr` (same behavior as [`lada qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `lada clawbot qr` -> `lada qr`
